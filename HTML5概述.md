html属性值是否一定要加引号？以下三种都可以，和命令行一样。
```html
<div id=aaa>aaa</div>
<div id='bbb'>bbb</div>
<div id="c c c">ccc</div>
```


闭合标签的结尾，不加/是最新规范
```html
<img src="a.png" />
<img src="a.png">
```
